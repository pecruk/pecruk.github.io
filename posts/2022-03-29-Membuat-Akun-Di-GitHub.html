Membuat Akun Di GitHub

Berikut ini adalah langkah Pecruk membuat akun di GitHub:

  • Ke https://github.com/.
  • Klik Sign up.
  • Cantumkan email kemudian klik Continue.
  • Cantumkan password kemudian klik Continue.
  • Cantumkan username kemudian klik Continue.
  • Ketik y untuk menjawab pertanyaan Would you like to receive product updates and announcements via email? kemudian klik Continue.
  • Selesaikan puzzle.
  • Klik Create account.
  • Cantumkan launch code yang terdapat pada email.
  • Selesaikan registrasi.
  • Klik Continue for free.
